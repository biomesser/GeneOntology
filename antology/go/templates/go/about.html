Вы читаете описание к небольшому django-проекту, где приведены условные идентификаторы генов, их rs_id и полигенные фенотипические признаки, которые могут обуславливаться данными генами.

Каждый ген связан с набором идентификаторов SNP по принципу "многие к одному", так как один ген содержит множество SNP, но отдельно взятый SNP может принадлежать только одному гену.

Для примера связи "один_к_одному" создана таблица с идентификаторами транскриптов. Здесь допущена условность, что каждый ген имеет только один транскрипт.

Гены и фенотипы связаны между собой по принципу "многие ко многим".

Функции сайта

Сайт выводит данные из базы данных в виде таблицы при помощи JS плагина DataTables и при помощи библиотеки django_tables2.
